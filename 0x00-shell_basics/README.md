# 0x00-shell_basics

This directory contains basic shell scripts for learning Linux commands.

## Scripts:
- `0-current_working_directory` - Prints the current working directory
- `1-listit` - Lists directory contents
- `2-bring_me_home` - Changes to home directory
- `3-listfiles` - Lists contents in long format
- `4-listmorefiles` - Lists all files (including hidden) in long format
